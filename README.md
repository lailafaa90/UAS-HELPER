# Kalkulus 2 UAS Helper

Website belajar Kalkulus 2 untuk UAS dengan materi, visualisasi, dan latihan soal dari kisi-kisi.

## Cara menjalankan

1. Buka folder ini di VS Code.
2. Install extension **Live Server**.
3. Klik kanan `index.html`.
4. Pilih **Open with Live Server**.

## Isi website

- Cara menentukan angka x dan batas integral.
- Logaritma natural.
- Integral parsial tentu dan tak tentu.
- Luas daerah.
- Metode cakram.
- Metode cincin.
- Metode kulit tabung.
- Visualisasi grafik dan daerah arsiran.
- Latihan soal lengkap dengan pembahasan langkah demi langkah sampai hasil akhir.

Catatan: rumus matematika menggunakan MathJax, jadi sebaiknya website dibuka dengan koneksi internet agar rumus tampil rapi.
